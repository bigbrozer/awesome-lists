    Kubernetes¶

    Authentication¶

    • kubelogin — Plugin for Kubernetes OpenID Connect authentication.

    Backup¶

    Cleaning¶

    • kube-janitor — Clean up (delete) Kubernetes resources after a configured TTL (time to live).

    Costing¶

    • kube-opex-analytics — Usage Analytics and Accounting for Cost Allocation and Capacity Planning. Limited free edition.

    Debugger¶

    • buildg — Interactive debugger for Dockerfile, with support for IDEs (VS Code, Emacs, Neovim, etc.)

    Development¶

    • gefyra — Blazingly-fast rocket, rock-solid, local application development with Kubernetes.

    • crd-ref-docs — Generates Kubernetes CRD API reference documentation.

    Exams¶

    • Kubernetes Certified Administrator — Online resources that will help you prepare for taking the CNCF CKA 2020 "Kubernetes Certified Administrator" Certification exam.

    Inventory¶

    • kubectl-images — Show container images used in the cluster.

    Monitoring¶

    • pixie — Instant Kubernetes-Native Application Observability

    • pyrra — Making SLOs with Prometheus manageable, accessible, and easy to use for everyone!

    Networking¶

    • kube-vip [homepage] — Provides Kubernetes clusters with a virtual IP and load balancer for both the control plane (for building a highly-available cluster) and Kubernetes Services of type LoadBalancer without relying on any external hardware or software.

    • k8spacket — helps to understand TCP packets traffic in your kubernetes cluster with packets traffic visualization.

    Security¶

    • Hardening Your Kubernetes Cluster — Covers the NSA and CISA guide on Kubernetes hardening in a serie of articles.
    • Part 1

    • Part 2

    • ThreatMapper — Open source cloud native security observability platform. Linux, K8s, AWS Fargate and more.

    • falco — Cloud Native Runtime Security

    • stackrox — The StackRox Kubernetes Security Platform performs a risk analysis of the container environment, delivers visibility and runtime alerts, and provides recommendations to proactively improve security by hardening the environment.

    Operators¶

    • config-syncer — Keeps ConfigMaps and Secrets synchronized across namespaces and/or clusters.

    Tests¶

    • kuttl — provides a declarative approach to test Kubernetes Operators and any kubernetes objects.
